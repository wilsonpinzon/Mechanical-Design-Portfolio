# Mechanical Design Portfolio

This repository contains a collection of mechanical design projects developed using CAD tools, primarily **Autodesk Inventor**. The portfolio showcases different mechanical components, assemblies, and engineering concepts created through academic projects and personal design work.

The purpose of this repository is to present practical experience in **mechanical design, 3D modeling, and technical documentation**, demonstrating the process from concept development to complete mechanical assemblies and technical drawings.

## What you will find in this repository

- **3D CAD models** of mechanical components and assemblies  
- **Engineering drawings** prepared for manufacturing and documentation  
- **Rendered images and visualizations** of mechanical systems  
- **Design concepts and prototypes** developed for different applications  

These projects reflect the application of **mechanical engineering principles**, including mechanical design, system integration, and design for manufacturing.

## Software Used

- Autodesk Inventor  
- AutoCAD
